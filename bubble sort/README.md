# Bubble Sort

**Time Complexity :**

- **In Worst Case :**  O(n^2)
- **In Best Case :**  O(n)
- **In Average Case :**  O(n^2)

**Space Complexity :**

Space Complexity is O(1)

**NOTE :**

In each iteration based on the condition ( accesending/desending order ) maximum/minimum element is placed at the end of the array.
